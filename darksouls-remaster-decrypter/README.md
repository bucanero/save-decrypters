# Dark Souls Remastered Decrypter

A tool to decrypt Dark Souls Remastered (PS4) save files

```
USAGE: ./darksouls-remastered-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### MD5 Hash

**Note:** this tool also updates the `MD5` integrity hash.
