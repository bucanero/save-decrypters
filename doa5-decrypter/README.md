# Dead or Alive 5 Save Decrypter

A tool to decrypt Dead or Alive 5 PS3, PS4, and PS Vita save-games.

```
USAGE: ./doa5-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Checksum

This tool also updates the custom ADD checksums.
