# ps4-re4r-decrypter

A tool to decrypt Resident Evil 4 Remake savegames.

```
USAGE: ./ps4-re4r-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
