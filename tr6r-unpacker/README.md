# tr6r-unpacker

A tool to unpack Tomb Raider VI: Angel of Darkness (PS4) save-games.

```
USAGE: ./tr6r-unpacker [option] filename

OPTIONS        Explanation:
 -u            Unpack File
 -p            Pack File
```

### Credits

The LZW compression is based on [TRR-SaveMaster](https://github.com/JulianOzelRose/TRR-SaveMaster) by [JulianOzelRose](https://github.com/JulianOzelRose)
