# dw8xl-decrypter

A tool to decrypt Dynasty Warriors 8 Xtreme Legends PS3 save-games

```
USAGE: ./dw8xl-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Credits

This tool is based (reversed) on the original `DW8XL Decryptor 1.1` by chaoszage
