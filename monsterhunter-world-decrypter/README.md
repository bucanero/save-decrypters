# mhw-decrypter

A tool to decrypt Monster Hunter World PS4 save-games.

```
USAGE: ./mhw-ps4-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Hash

**Note:** the tool also updates the custom SHA1 integrity hash.
