# Borderlands 3 PS4 save decrypter

A tool to decrypt Borderlands 3 PS4 save-games (Profile and Saves)

```
USAGE: ./ps4-borderlands3-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
