# Final Fantasy Pixel Remaster PS4 Save Decrypter

A tool to decrypt Final Fantasy Pixel Remaster PS4 save-games

```
USAGE: ./ff-pixel-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
