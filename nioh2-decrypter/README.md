# Nioh 2 (PS4) save decrypter

A tool to decrypt/encrypt Nioh 2 (PS4) save-games.

```
USAGE: ./nioh2-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Checksum

Note: this tool doesn't update the custom checksum, but disables the integrity checks by editing flags on the game's latest version save data.
