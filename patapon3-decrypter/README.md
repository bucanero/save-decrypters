# Patapon 3 save decrypter

A tool to decrypt Patapon 3 PSP save-games.

```
USAGE: ./patapon3-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Credits

This tool is based on [libP3Hash](https://github.com/efonte/libP3Hash) by Owocek.
