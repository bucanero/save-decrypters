# L.A. Noire PS3/PS4 Save Decrypter

A tool to decrypt L.A. Noire PS3 and PS4 save-games

```
USAGE: ./la-noire-decrypter [option] [type] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
TYPE
 -1            SaveData (PS3)
 -2            Profile  (PS3)
 -3            SaveData (PS4)
 -4            Profile  (PS4)
```
