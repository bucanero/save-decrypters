# L.A. Noire PS3 Save Decrypter

A tool to decrypt L.A. Noire PS3 save-games

```
USAGE: ./la-noire-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
