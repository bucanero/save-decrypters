# Tales of Berseria Save Decrypter

A tool to decrypt Tales of Berseria (PS3/PS4/PC) save-games.

```
USAGE: ./berseria-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Hash

**Note:** this tool also updates the custom SHA1 integrity hash.
