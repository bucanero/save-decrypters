# nfsrivals-ps3save-decrypter

A tool to decrypt Need For Speed: Rivals PS3 save-games. The tool also fixes the custom CRC32 integrity hash.

```
USAGE: ./nfsrivals-ps3save-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```
