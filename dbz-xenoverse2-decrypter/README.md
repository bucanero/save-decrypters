# DBZ Xenoverse 2 Decrypter

A tool to decrypt DBZ Xenoverse 2 (PS4) save files

```
USAGE: ./dbz-xenoverse2-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### MD5 Hash

**Note:** this tool also updates the `MD5` integrity hash.
