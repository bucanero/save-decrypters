# Crisis Core Final Fantasy VII PS4 Save Decrypter

A tool to decrypt Crisis Core: Final Fantasy VII PS4 save-games

```
USAGE: ./ff7cc-decrypter [option] filename

OPTIONS        Explanation:
 -d            Decrypt File
 -e            Encrypt File
```

### Checksum

**Note:** this tool also updates the custom integrity checksum.
