# ps4-nms-unpacker

A tool to unpack No Man's Sky PS4 `savedata.hg` files.

```
USAGE: ./ps4-nms-unpacker [option] filename

OPTIONS        Explanation:
 -u            Unpack File
 -p            Pack File
```

### Notes

- Only `savedata.hg` files are compatible
